查看資料庫 →

點擊下方按鈕，後端會用 requests 去抓外部 API 的資料， 再存進本機的 SQLite 資料庫。 每按一次就會新增一筆紀錄。

🌐

抓取 Web API

向 JSONPlaceholder 請求一篇示範文章（/posts/1）， 並將標題與內容存入資料庫。

GET jsonplaceholder.typicode.com/posts/1
▶ 執行抓取
🐙

抓取 GitHub 資料

向 GitHub API 請求 Google 官方帳號的公開資訊， 並將名稱與簡介存入資料庫。

GET api.github.com/users/google
▶ 執行抓取
🔗

開放 API 端點

你的 Flask 伺服器也能對外提供資料！ 鄰座同學可以用 client.py 來抓你的資料。

GET 127.0.0.1:5000/api/posts
▶ 查看 JSON
